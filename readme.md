# 📂 File Upload Backend

A backend service for uploading, storing, parsing, and retrieving files with real-time progress tracking.

# 🛠 Tech Stack

* Node.js + Express
* MongoDB (Mongoose)
* Multer (file uploads)
* UUID (unique file IDs)
* dotenv (env variables)
